## 🔍 What This Project Does

Transforms raw e-commerce transaction data into actionable customer intelligence:

1. **Segments customers** into 4 behavioural groups using RFM + KMeans
2. **Predicts churn probability** per customer using Logistic Regression
3. **Recommends retention actions** based on segment × churn risk
4. **Serves predictions via REST API** (FastAPI) and a live dashboard (Streamlit)

## 🚀 Quick Start

### Option A — Docker (Recommended)

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/customer-churn-app.git
cd customer-churn-app

# 2. Train the models (needs online_retail_II.xlsx in root)
pip install -r requirements.txt
python training/train.py

# 3. Launch both services
docker compose up --build

# API docs:      http://localhost:8000/docs
# Dashboard:     http://localhost:8501
```

### Option B — Local (No Docker)

```bash
pip install -r requirements.txt

# Train
python training/train.py

# Run API (terminal 1)
uvicorn api.main:app --reload --port 8000

# Run frontend (terminal 2)
streamlit run frontend/app.py
```

## 📡 API Reference

### `POST /predict` — Single customer
```json
{
  "customer_id": "C12345",
  "recency": 30,
  "frequency": 15,
  "monetary": 850.00
}
```
**Response:**
```json
{
  "customer_id": "C12345",
  "segment": "High-Value Loyal",
  "cluster_id": 1,
  "churn_probability": 0.12,
  "churn_risk": "Low",
  "retention_action": "Monitor"
}
```

### `POST /predict/batch` — Up to 500 customers
```json
{
  "customers": [
    { "customer_id": "C001", "recency": 10, "frequency": 25, "monetary": 1500 },
    { "customer_id": "C002", "recency": 120, "frequency": 3, "monetary": 200 }
  ]
}
```

## 📊 Customer Segments

| Segment | RFM Profile | Retention Strategy |
|---|---|---|
| 🏆 High-Value Loyal | Low R · High F · High M | VIP programme, early access |
| ⚠️ At-Risk High-Value | High R · High M | Personalised win-back |
| 📈 Active Low-Value | Low R · Low M | Upsell / bundle offers |
| 💤 Inactive / Churned | Very High R · Low F · Low M | Low-cost re-engagement |

## ☁️ Deploy to Render (Free)

1. Push to GitHub
2. Create a new **Web Service** on [render.com](https://render.com)
3. Set **Build Command**: `pip install -r requirements.txt`
4. Set **Start Command**: `uvicorn api.main:app --host 0.0.0.0 --port $PORT`
5. Add env var: `MODEL_DIR=./models`
6. Deploy ✅
